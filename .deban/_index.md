# kakistogenesis — Index

## Brief
Interactive visual reference cataloguing eleven institutional mechanisms — across self-preservation, capture, and emergence categories — by which mixed-intent input becomes asymmetrically negative output. Each mechanism carries academic provenance (originator, year, citation, link). Canonical name: **ペシ魔鉄則 (Peshi-ma Tessoku)** — Latin *pessimum* + 魔 (demon, the asymmetric exploiter) + 鉄則 (iron rule). Five alternative framings (Mens Bona Mala Eventa, Kakistogenesis, Negative Maelstrom, Corruptio Optimi Pessima, Kakisto/Pessimum Filter) live in a discussion tab as register options. Current state: 50KB PoC `kakistogenesis.jsx` exists as reference. V1 build is a blank-slate static animated PWA dispatched to a PM agent: Vite + React + Tailwind + framer-motion + vite-plugin-pwa, with the Filter Chamber rendered as an elegant whirling maelstrom that absorbs rainbow good-intent particles and emits uniformly black bad-outcome flow.
